# LynkDrive Landing Page

Landing page built with Vite, React, TypeScript, Tailwind CSS, and shadcn-ui.

## Requirements

- Node.js 18+
- npm (or bun)

## Local development

```sh
npm install
npm run dev
```

The dev server runs at http://localhost:8080.

## Build

```sh
npm run build
npm run preview
```
