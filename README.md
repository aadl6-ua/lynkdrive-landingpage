# LYNKDRIVE Landing Page

Landing page premium y accesible para LYNKDRIVE, plataforma P2P europea de alquiler de vehículos. Incluye todas las secciones del briefing, copy real y animaciones suaves.

## Tecnologías

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Framer Motion

## Desarrollo local

```bash
npm run dev
```

La página principal está en `app/page.tsx`.

## Producción

```bash
npm run build
npm run start
```

## Notas

- Los bloques marcados como placeholders de imagen deben reemplazarse por assets reales.
